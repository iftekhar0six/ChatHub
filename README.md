# ChatHub

![GitHub](https://img.shields.io/github/license/iftekhar0six/ChatHub)

ChatHub is a real-time chat application built with Node.js, Express, and Socket.io. It allows users to engage in live conversations with each other.

- [Installation](#installation)
- [Usage](#usage)
- [Features](#features)

## Installation

Before you get started, ensure that you have [Node.js](https://nodejs.org/) installed on your system.

1. Clone this repository:

   ```bash
   git clone https://github.com/iftekhar0six/ChatHub.git
   ```

2. Navigate to the server folder:

   ```bash
   cd ChatHub/Nodeserver
   ```

3. Install the required Node.js packages:

   ```bash
   npm install nodemon server cors express socket.io
   ```

## Usage

To run the ChatHub server, make sure your terminal is open in the `ChatHub/Nodeserver` folder and execute the following command:

```bash
npm start
```

The server will start and run at [http://127.0.0.1:5501/](http://127.0.0.1:5501/).

Once the server is running, you can access the chat application from your web browser at this address.

- Real-time chat with multiple users
- User joining and leaving notifications
